INSTALLATION
============
1. Upload the entire contents of this folder to your hosting (e.g. public_html/shop/)
2. Ensure data/ and uploads/ folders are writable (chmod 755 or 775)
3. Open https://your-domain.com/shop/ in your browser — demo products and settings load from JSON
4. Admin panel: https://your-domain.com/shop/admin/
   Demo login: demo / demo2026

REQUIREMENTS
============
- PHP 8.0+
- Writable data/ and uploads/ folders
- No database required

DATA STORAGE
============
Products, categories, news, settings and leads are stored in JSON files under data/.
Edit content via admin panel or replace JSON files before upload.

AI FEATURES
=============
Configure API key in Admin → Settings → AI. Without a key, demo templates are applied locally.
