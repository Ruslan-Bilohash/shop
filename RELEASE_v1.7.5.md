## Shop CMS v1.7.5

### Import / Export — store migration
- **Migration hub** on `admin/products-io.php` — one-click setup for WooCommerce, Shopify, OpenCart, PrestaShop, Rozetka, Prom.ua, Magento and generic CSV
- Step-by-step migration guide per platform with smooth scroll to import form
- **Import auto-detect** — reads CSV headers and picks the best parser (Shopify, WooCommerce, Prom.ua, etc.)
- **File preview** — count products and detected format before writing to catalog
- Advanced import: preserve SEO, skip images, update prices/stock only, merge / append / replace modes

### New export formats
- **Shopify**, **PrestaShop**, **Magento / Adobe Commerce**, **Prom.ua** (marketplace CSV)
- **Google Merchant Center** and **Facebook / Meta Catalog** product feeds
- Export filters: category, active, featured, in stock; optional SEO in JSON backup

### i18n
- Full `products_io_page` translations: NO, EN, UK, RU, SV, LT

### Packages
- `shop-install-v1.7.5.zip` — MySQL commercial package
- `shop-not-mysql-v1.7.5.zip` — JSON storage edition
- `shop-demo-30d-v1.7.5.zip` — 30-day demo package (MySQL install wizard)
